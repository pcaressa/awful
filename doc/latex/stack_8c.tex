{\footnotesize\ttfamily \#define CHUNKSIZ~(1024)}

A chunk is an array of CHUNKSIZ stack items\+: each stack in the program takes its items from a chunk. The function \mbox{\hyperlink{stack_8c_a40264fa48696006f6219eb5731d5aa98}{stack\+\_\+reset()}} is used to free all chunks for future reuse.
