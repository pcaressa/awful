\doxysubsubsection{\texorpdfstring{repl()}{repl()}}
{\footnotesize\ttfamily int repl (\begin{DoxyParamCaption}\item[{FILE $\ast$}]{in,  }\item[{FILE $\ast$}]{out,  }\item[{int($\ast$)(char $\ast$, FILE $\ast$)}]{interpret,  }\item[{char $\ast$}]{prompt }\end{DoxyParamCaption})}

Activates a REPL by applying the interpret function to each string introduced by the user. Input lines are read from the in file, and results and error messages are printed on the out file. If not NULL, the prompt is printed each time a line is read from in. The interpret function should have signature int interpret(char$\ast$,\+FILE$\ast$) it takes a string in input, evaluates it and prints the result (or an error message) on the file.
