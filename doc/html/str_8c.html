◆ TABSIZE

#define TABSIZE   (1024) /* Need to be a power of 2 */

Strings are stored into a hash table whose elements are stack items: each item of the stack contains a string whose hash encoding is the same.

When a string is created, it is inserted or retrieved from the table.

char* str_strip ( char *  s)

Return the address of the substring of s which is stripped by spaces, both on the left and on the right: on the left the effect is achieved returing a pointer on the first non space character, on the right by setting to '\0' the leftmost space from the end of the string.
