Activates a REPL by applying the interpret function to each string introduced by the user. Input lines are read from the in file, and results and error messages are printed on the out file. If not NULL, the prompt is printed each time a line is read from in. The interpret function should have signature int interpret(char*,FILE*) it takes a string in input, evaluates it and prints the result (or an error message) on the file.
