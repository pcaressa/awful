static

Gets a line from a file. If prompt != NULL then it is printed on repl_out: the scanned line is stored at repl_buf if cat == 0, else it is appended to the string already in repl_buf. In any case, the address of the last inserted line is returned; if an error, or the end of the file, occurs, NULL is returned.
